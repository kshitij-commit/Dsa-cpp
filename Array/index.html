Array

An Array is a fixed size sequence collection of elements of same data type

Creating an Array

int[] numbers = new int[5]; // Creates an array of integers with a length of 5

Accessing Array Elements

numbers[0] = 10; // Assigns the value 10 to the first element of the array
int firstNumber = numbers[0]; // Retrieves the value of the first element

Array Length

int length = numbers.Length; // Gets the length of the array
